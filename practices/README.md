# 正式练习结构

`practices/` 是新正式练习的目标目录。练习按使用场景分类，不按 HTML、小游戏或 quiz 等技术形态分类。

```text
practices/
├─ classroom/  随堂挑战
├─ special/    专项练习
└─ grammar/    语法挑战
```

## 分类边界

- `classroom/`：与某次课堂或日期绑定、由老师使用的即时挑战。
- `special/`：围绕某一项能力反复练习，不依赖某次课堂。
- `grammar/`：围绕语法知识组织的系统挑战。

主应用里的“今日挑战 / 混合挑战”属于任务系统，不迁入这里。

## 兼容策略

- 现有 `courseware/` 保存历史随堂练习 URL，本轮不移动；产品名称统一理解为“随堂挑战”。
- 现有 `quizzes/` 保存已经上线的专项小游戏，本轮不移动。
- 新模板统一放在 `practice-templates/`，不得登记为正式练习。
- 后续迁移时先更新注册表并保留旧路径兼容，不能直接破坏历史链接。
